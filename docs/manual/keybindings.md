# Keybindings

lazydb のキーバインド一覧です。Vim風の操作を採用しています。

## グローバル

| キー | 説明 |
|------|------|
| `q` | アプリケーションを終了 |
| `Ctrl+c` | アプリケーションを終了 |
| `Tab` | 次のパネルへフォーカス移動 |
| `Shift+Tab` | 前のパネルへフォーカス移動 |

## ナビゲーション

| キー | 説明 |
|------|------|
| `j` / `↓` | 下へ移動 |
| `k` / `↑` | 上へ移動 |
| `Shift+h` / `Shift+←` | 左のパネルへ移動 |
| `Shift+l` / `Shift+→` | 右のパネルへ移動 |
| `Shift+k` / `Shift+↑` | 上のパネルへ移動 |
| `Shift+j` / `Shift+↓` | 下のパネルへ移動 |
| `Enter` | 選択/実行 |

## サイドバー（プロジェクト・接続リスト）

サイドバーには「プロジェクト」モードと「接続」モードの2つがあります。

- **プロジェクトモード**: プロジェクト一覧が表示されます
- **接続モード**: 選択したプロジェクトの接続一覧が表示されます

| キー | 説明 | モード |
|------|------|------|
| `Enter` | プロジェクトを選択して接続一覧へ | プロジェクト |
| `Backspace` | プロジェクト一覧に戻る | 接続 |
| `Enter` | 接続の展開/折りたたみ | 接続 |
| `a` | 接続を追加（モーダルを開く） | 接続 |

## メインパネル

| キー | 説明 |
|------|------|
| `s` | スキーマタブに切り替え |
| `d` | データタブに切り替え |

## モーダルダイアログ

接続追加モーダルなど、モーダルが開いている時のキーバインドです。

### フィールド間の移動

| キー | 説明 |
|------|------|
| `Tab` | 次のフィールドへ |
| `Shift+Tab` | 前のフィールドへ |
| `Enter` | 次のフィールドへ（ボタン上では実行） |

### ボタン上での操作

| キー | 説明 |
|------|------|
| `h` / `←` | 前のボタンへ |
| `l` / `→` | 次のボタンへ |
| `j` / `↓` | 次のボタンへ |
| `k` / `↑` | 前のボタンへ |
| `Enter` | ボタンを実行 |

### その他

| キー | 説明 |
|------|------|
| `Esc` | モーダルを閉じる（キャンセル） |
| `Backspace` | 1文字削除 |

## クエリ履歴

| キー | 説明 |
|------|------|
| `Ctrl+r` | クエリ履歴モーダルを開く |
| `j` / `↓` | 次の履歴へ |
| `k` / `↑` | 前の履歴へ |
| `Enter` | 選択した履歴をクエリエディタに挿入 |
| `c` | 履歴をクリア |
| `Esc` / `q` | モーダルを閉じる |
