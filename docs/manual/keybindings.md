# Keybindings

lazydb のキーバインド一覧です。Vim風の操作を採用しています。

## グローバル

| キー | 説明 |
|------|------|
| `q` | アプリケーションを終了 |
| `Ctrl+c` | アプリケーションを終了 |
| `Tab` | 次のパネルへフォーカス移動 |
| `Shift+Tab` | 前のパネルへフォーカス移動 |

## ナビゲーション

| キー | 説明 |
|------|------|
| `j` / `↓` | 下へ移動 |
| `k` / `↑` | 上へ移動 |
| `Enter` | 選択/実行 |

## サイドバー（接続リスト）

| キー | 説明 |
|------|------|
| `Enter` | 接続の展開/折りたたみ |
| `a` | 接続を追加（モーダルを開く） |

## メインパネル

| キー | 説明 |
|------|------|
| `s` | スキーマタブに切り替え |
| `d` | データタブに切り替え |

## モーダルダイアログ

接続追加モーダルなど、モーダルが開いている時のキーバインドです。

### フィールド間の移動

| キー | 説明 |
|------|------|
| `Tab` | 次のフィールドへ |
| `Shift+Tab` | 前のフィールドへ |
| `Enter` | 次のフィールドへ（ボタン上では実行） |

### ボタン上での操作

| キー | 説明 |
|------|------|
| `h` / `←` | 前のボタンへ |
| `l` / `→` | 次のボタンへ |
| `j` / `↓` | 次のボタンへ |
| `k` / `↑` | 前のボタンへ |
| `Enter` | ボタンを実行 |

### その他

| キー | 説明 |
|------|------|
| `Esc` | モーダルを閉じる（キャンセル） |
| `Backspace` | 1文字削除 |
